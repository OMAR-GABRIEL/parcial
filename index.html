Documentación del Código Python para el Modelo de TensorFlow

Este código entrena un modelo de red neuronal simple para predecir valores de temperatura en Fahrenheit a partir de valores en Celsius utilizando TensorFlow. A continuación se explica cada línea del código:


# Importa la biblioteca TensorFlow, que se utiliza para construir y entrenar redes neuronales.
import tensorflow as tf  

# Importa NumPy, que proporciona soporte para arrays multidimensionales.
import numpy as np  

# Define un array de temperaturas en grados Celsius como datos de entrada para el modelo.
celsius = np.array([-40, 100, 8, 15, 22, 38], dtype=float)

# Define un array de temperaturas en grados Fahrenheit como datos objetivo para el modelo.
fahrenheit = np.array([-40, 100, 8, 15, 22, 38], dtype=float)

# Crea una capa densa (fully-connected) con 3 unidades (neuronas) y especifica que la entrada tendrá 1 valor (un número).
ocular1 = tf.keras.layers.Dense(units=3, input_shape=[1])

# Crea otra capa densa con 3 neuronas, sin especificar explícitamente la forma de entrada.
ocular2 = tf.keras.layers.Dense(units=3)

# Define la capa de salida con 1 unidad, ya que la salida será un valor único (temperatura en Fahrenheit).
salida = tf.keras.layers.Dense(units=1)

# Define un modelo secuencial compuesto por las dos capas ocultas y la capa de salida.
modelo = tf.keras.Sequential([ocular1, ocular2, salida])

# Sobrescribe el modelo anterior con una capa densa con 1 unidad y una entrada de 1 valor.
capa = tf.keras.layers.Dense(units=1, input_shape=[1])

# Define un nuevo modelo secuencial con una sola capa.
modelo = tf.keras.Sequential([capa])

# Compila el modelo con el optimizador Adam (tasa de aprendizaje de 0.1) y la pérdida de error cuadrático medio.
modelo.compile(
    optimizer=tf.keras.optimizers.Adam(0.1),
    loss='mean_squared_error'
)

# Muestra un mensaje indicando que el entrenamiento está por comenzar.
print("comenzando entrenamiento")

# Entrena el modelo con los datos de Celsius y Fahrenheit durante 1000 épocas, sin mostrar la información de entrenamiento.
historial = modelo.fit(celsius, fahrenheit, epochs=1000, verbose=False)

# Muestra un mensaje indicando que el modelo ha terminado de entrenarse.
print("modelo entrenado")

# Importa la biblioteca Matplotlib para generar gráficos.
import matplotlib.pyplot as plt

# Etiqueta el eje X del gráfico como 'epoca'.
plt.xlabel("# epoca")

# Etiqueta el eje Y del gráfico como 'Magnitud de perdida'.
plt.ylabel("Magnitud de perdida")

# Grafica la pérdida (loss) del modelo durante el entrenamiento.
plt.plot(historial.history["loss"])

# Muestra un mensaje indicando que se va a realizar una predicción.
print("hagamos una predeccion")

# Realiza una predicción para un valor de 100 grados Celsius.
resultado = modelo.predict(np.array([100.0]))

# Imprime el resultado de la predicción en Fahrenheit.
print("el resultado es " + str(resultado) + " fahrenheit!")

# Muestra un mensaje indicando que se va a imprimir la variable interna del modelo (los pesos de la capa).
print("variable interna del modelo")

# Muestra los pesos de la capa que el modelo ha aprendido durante el entrenamiento.
print(capa.get_weights())
